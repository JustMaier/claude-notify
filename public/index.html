API Usage

Send notifications via POST request:

POST /notify
Content-Type: application/json

{
  "title": "Claude Done",
  "body": "Task completed successfully",
  "url": "/"
}

curl example:

curl -X POST http://localhost:3939/notify \
  -H "Content-Type: application/json" \
  -d '{"title":"Claude","body":"Done!"}'
